### Project Description

Spots is an interactive image-sharing platform where users can post and view photos of interesting locations. This stage of the project focused on making the application fully **responsive** and **adaptive** for all screen sizes.

### Technologies and Techniques

- **Responsive Design**: Used Flexbox and CSS Grid (with `auto-fit`) to create a layout that adjusts from desktop to mobile.
- **Media Queries**: Implemented at `630px` to handle mobile layouts.
- **Text Truncation**: Used CSS properties like `text-overflow: ellipsis` and `-webkit-line-clamp` to keep the UI clean by hiding overflowing text.
- **Adaptive Assets**: Images and containers scale using `max-width` and relative units.

### Features

- **Grid Layout**: 3 columns on desktop, 2 on tablet, and 1 on mobile.
- **Profile Styling**: Elements center and stack vertically on smaller screens.
- **Hover Effects**: Interactive buttons and links change state for better UX.
